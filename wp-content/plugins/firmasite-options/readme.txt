This plugin provides theme display options for users & visitors.
== Description ==

Plugin is working only when [FirmaSite](http://wordpress.org/themes/firmasite) theme activated.
Live Demo: [http://theme.firmasite.com](http://theme.firmasite.com)

Its also useful for testing your custom design changes & custom bootstrap skins
